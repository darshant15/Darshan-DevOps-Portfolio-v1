# [ darshan@devops ] — Portfolio

A terminal-themed DevOps portfolio website built with pure HTML, CSS, and JavaScript.

## 📁 Project Structure

```
darshan-portfolio/
├── index.html          ← Main entry point
├── README.md           ← You are here
└── assets/
    ├── style.css       ← All styles (light terminal theme)
    ├── script.js       ← Animations, typing effect, interactive terminal
    ├── photo.jpg       ← Profile photo
    └── resume.pdf      ← Resume (add your own PDF here)
```

## 🚀 Deploy

### GitHub Pages (free)
1. Push this folder to a GitHub repo
2. Go to Settings → Pages → Source: main branch
3. Your site is live at `https://yourusername.github.io/darshan-portfolio`

### Netlify (free, instant)
1. Go to netlify.com
2. Drag and drop this entire folder
3. Done — live URL in seconds

### Vercel (free)
```bash
npm i -g vercel
cd darshan-portfolio
vercel
```

## ✏️ Customise

- **Your info** → edit `index.html` (name, bio, experience, projects, links)
- **Colors/fonts** → edit `assets/style.css` (CSS variables at the top of the file)
- **Animations/terminal** → edit `assets/script.js`
- **Photo** → replace `assets/photo.jpg` with your own (keep same filename)
- **Resume** → add your PDF as `assets/resume.pdf`

## ⚙️ Tech Stack

- Pure HTML5 / CSS3 / Vanilla JavaScript
- No frameworks, no build tools, no dependencies
- Google Fonts: JetBrains Mono + Inter
- Works in all modern browsers
